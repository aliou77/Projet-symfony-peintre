### site peintre

site peintre est un site web qui presente des peintures

### Environement de developpement
## pre-requis

- PHP 8
- Maildev (nodeJS)
- Composer
  
### Lancer l'environement de developpement

``` bash
php -S localhost:8080 -t public 
```

## Lancer des tests

``` bash
php bin/phpunit --testdox  
```
